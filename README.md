# Healthcare-Disease-Prediction

## Domain
Healthcare / Medical Data Analytics

## Objective
To predict the presence of disease in patients using clinical parameters.

## Dataset
Features:
- Age
- Blood Pressure
- Cholesterol
- BMI
- Glucose

Target:
- Disease (0 = No Disease, 1 = Disease)

## Algorithm Used
Logistic Regression

## Steps to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run the program:
   python disease_prediction.py

## Output
- Accuracy score
- Confusion matrix

## Applications
- Early disease detection
- Decision support for doctors
- Preventive healthcare analytics
